Lineage

Inspired by Atom and Pulsar, maintained independently.

Lumine continues the idea of an editor you can take apart and rebuild around your workflow, with the project stewarded by the community that uses it.

Build

Build from source

Lumine runs from source: clone the repo, use the Node version from .nvmrc, then install and start. npm ships with Node, so there is nothing else to set up.

Recommended local flow

Runs the app from source 
# Clone the source
git clone https://github.com/lumine-code/lumine.git
cd lumine

# Use the project Node version from .nvmrc
nvm install

# Install, rebuild native modules, and launch
npm install
npm run build
npm start

# Optional: produce platform binaries in binaries/
npm run dist

Dev resource path

If Lumine is already installed, point the installed app at this source checkout and launch it in dev mode.

# PowerShell
$env:LUMINE_RESOURCE_PATH = (Get-Location).Path
lumine --dev .

# macOS/Linux
LUMINE_RESOURCE_PATH="$PWD" lumine --dev .
